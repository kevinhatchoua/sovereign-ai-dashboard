# Security Policy

## Supported Versions

We only provide security updates for the current `main` branch.

## Reporting a Vulnerability

Please do not report security vulnerabilities via public GitHub issues. Instead, email [Your Email] or open a [private security advisory](https://github.com/kevinhatchoua/sovereign-ai-dashboard/security/advisories/new) on GitHub.

## Data Integrity

This dashboard uses public metadata. If you find data that is outdated or misleading, please use the **Report Compliance Dispute** feature in the app.
